三、集成架构

3.1 调用方式

Hermes 通过标准 HTTP 请求调用 Fincept API，与调用 iFind/Choice MCP 的方式完全一致：

# Hermes 调用 Fincept：期权定价
import requests
resp = requests.post(
    "https://api.fincept.in/quantlib/pricing/black-scholes",
    headers={"X-API-Key": "fk_user_xxx"},
    json={
        "spot": 100, "strike": 105,
        "rate": 0.05, "volatility": 0.2,
        "time_to_maturity": 0.25, "option_type": "call"
    },
    timeout=30
)
greeks = resp.json()["data"]
# {"price": 2.456, "delta": 0.543, "gamma": 0.028, ...}

3.3 封装为 Skill

将 Fincept API 封装成 Hermes 的 skill，像使用 iFind/Choice 一样自然调用：

# skill: fincept-quantlib
# 触发词: "计算期权Greeks" / "组合VaR" / "BS定价"

def call_fincept(endpoint, params):
    """统一的 Fincept API 调用入口"""
    return requests.post(
        f"https://api.fincept.in/{endpoint}",
        headers={"X-API-Key": API_KEY},
        json=params, timeout=30
    ).json()

# 使用示例
# Hermes: "帮我算一下铜期权的Delta和Gamma"
greeks = call_fincept("quantlib/pricing/black-scholes", {
    "spot": 75000, "strike": 76000, "rate": 0.03,
    "volatility": 0.22, "time_to_maturity": 0.25
})
