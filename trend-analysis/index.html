📋 均线支撑/压制 + 级联分析 — 方法说明

一、均线支撑/压制测试

在每个趋势段内，测试各均线(M20-MA233)的支撑或压制效果:

  • 触碰检测: 收盘价与MA偏离≤1.5%
  • 方向过滤: 上升趋势只看从上方触碰(测支撑)，下降趋势只看从下方触碰(测压制)
  • 成效确认: 触碰后5日内反弹/回落≥2% = 有效
  • 评分: 成功率 × log₁₀(触碰次数+1)，兼顾效果和统计意义

二、均线级联分析(逐级递推)

测试MA5失守后价格沿均线逐级寻找支撑/压制的行为:

  • 触发: 趋势中收盘价跌破/突破MA5
  • 反转识别: 向后逐日寻找最低/最高收盘价，之后反弹/回落≥2% = 反转
  • 承接均线: 反转点价格偏离≤5%的最近均线 = 止跌/止涨位
  • 权重分级: 反转后创新高/新低 = 明确反转(×2)；未创新高/新低 = 普通反转(×1)
  • 逐级递推: 若MA10未承接 → 检查MA20 → MA34 → ... → MA233
